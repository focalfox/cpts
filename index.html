Cybefox

Week 1 - Introduction to Penetration Testing

Day 1

Module: 1 - Penetration Testing Process

Focus: Understand the overall process of penetration testing, methodologies, and frameworks.

Tools: N/A

Resources: Course material, "The Pentester BluePrint" by Phillip L. Wylie

Study Time: 2 hours

Activities: Read the module, take notes, and understand the phases of a penetration test.

Day 2

Module: 2 - Getting Started

Focus: Set up your lab environment, understand the tools, and get familiar with basic commands.

Tools: VirtualBox, Kali Linux

Resources: Course material, Kali Linux Documentation

Study Time: 3 hours

Activities: Set up the lab, install necessary tools, practice basic Linux commands.

Day 3

Module: 3 - Network Enumeration with Nmap

Focus: Learn how to use Nmap for network enumeration.

Tools: Nmap

Resources: "Nmap Network Scanning" by Gordon Fyodor Lyon

Study Time: 2 hours

Activities: Read the module, practice Nmap commands on your lab environment.

Day 4

Module: 4 - Footprinting

Focus: Techniques for gathering information about the target.

Tools: Recon-ng, theHarvester

Resources: "The Web Application Hacker's Handbook" by Dafydd Stuttard, Marcus Pinto

Study Time: 2 hours

Activities: Practice footprinting techniques on various websites.

Day 5

Module: 5 - Information Gathering - Web Edition

Focus: Gathering information specifically for web applications.

Tools: Burp Suite, OWASP ZAP

Resources: OWASP Testing Guide

Study Time: 2 hours

Activities: Perform information gathering on sample web applications.

Day 6

Module: 6 - Vulnerability Assessment

Focus: Learn how to assess and identify vulnerabilities.

Tools: Nessus, OpenVAS

Resources: Nessus Documentation, "Mastering Modern Web Penetration Testing" by Prakhar Prasad

Study Time: 2 hours

Activities: Conduct vulnerability assessments on your lab environment.

Day 7

Module: 7 - File Transfers

Focus: Techniques for transferring files during a penetration test.

Tools: Netcat, SCP, FTP

Resources: Course material, Linux man pages

Study Time: 2 hours

Activities: Practice file transfer techniques between systems in your lab.

Module: 10 - Password Attacks

Day 11

Module: 11 - Attacking Common Services

Focus: Exploiting common network services.

Tools: Nmap, Metasploit

Resources: "Hacking: The Art of Exploitation" by Jon Erickson

Study Time: 2 hours

Day 12

Module: 12 - Pivoting, Tunneling, and Port Forwarding

Activities: Practice pivoting and tunneling in your lab environment.

Module: 13 - Active Directory Enumeration & Attacks

Day 14

Module: 14 - Using Web Proxies

Study Time: 2 hours

Activities: Set up and use web proxies for interception and modification.
